## Welcome

This document guides the installation and configuration of our Minecraft Modpacks using BisectHosting and SimplisticNode. It outlines the basic setup process, supported configurations, and required steps to ensure proper server deployment and operation. By following this guide, users acknowledge responsibility for their server environment and agree to adhere to all applicable hosting provider policies and Darkmoon Hollow guidelines.

## [BisectHosting](bisecthosting.com)

BisectHosting is a game server hosting provider specializing in Minecraft servers, offering user-friendly control panels, one-click modpack installations, automatic backups, and reliable performance. It is well-suited for both beginners and advanced users looking for stable hosting with flexible configuration options.

## [SimplisticNode](https://simplisticnode.com/)

SimplisticNode is a game server hosting provider focused on performance, customization, and affordability, offering full file access, configurable server environments, and support for modded and custom Minecraft setups. It is ideal for users who prefer greater control over server configuration and deployment.
